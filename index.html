How to Play Vaccine Jeopardy

Objective:

Answer questions correctly to earn points and become the Vaccine Jeopardy champion!

Gameplay:

  • Choose a category and point value from the game board
  • Read the question and select your answer within 30 seconds
  • Correct answers earn the point value shown
  • Incorrect answers deduct the point value
  • Try to answer as many questions as you can to maximize your score
